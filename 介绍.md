# 通用弹窗封装说明

项目内基于 Fancybox v5 封装了两个独立的弹窗模块，使用方式都是「只在 HTML 上加属性，零 JS」。

---

## 一、图集弹窗（image-popup）

文件：`css/image-popup.css` + `js/image-popup.js`

`js/image-popup.js` 已经做成全局绑定 `[data-fancybox]`，要给其他图片加放大功能不需要写任何 JS，只需要在 HTML 上把对应 `<img>` 包一层 `<a>` 即可：

```html
<a data-fancybox="项目分组名" href="大图路径.jpg" data-caption="可选说明文字">
    <img src="缩略图.jpg" alt="说明">
</a>
```

注意点：

- `data-fancybox` 的值要同区块同名才会被归到同一相册（左右翻页只在同名分组内）
- 不同区块之间要避免重名，否则会串到一个相册里
- `data-caption` 可选；不写时会自动取 `<img alt>` 或周围的 `.xx_label` / `.xx_caption` 文本

---

## 二、视频弹窗（video-popup）

文件：`css/video-popup.css` + `js/video-popup.js`

`js/video-popup.js` 已经做成全局绑定 `[data-video-popup]`，无需写任何 JS。视频弹窗为单视频播放，**没有左右切换、没有缩略图、没有计数器**，右上角深色条与图集弹窗一致保留 [下载] [关闭] 两个按钮，并支持 ESC 键关闭。

最简用法：

```html
<a data-video-popup href="videos/project.mp4" aria-label="播放视频"></a>
```

带封面图 + 标题：

```html
<a data-video-popup href="videos/project.mp4" data-caption="项目视频">
    <img src="cover.jpg" alt="项目封面">
    <span class="play-icon"></span>
</a>
```

支持的视频源（按 `href` 自动识别）：

| 类型 | 示例 |
| --- | --- |
| HTML5 视频 | `href="videos/xx.mp4"`、`.webm`、`.ogv` |
| YouTube / Vimeo | `href="https://www.youtube.com/watch?v=..."` |

注意点：

- 每个 `[data-video-popup]` 都会被自动赋予唯一分组值，**默认互不干扰、互不切换**；不需要像 `data-fancybox` 那样手动起分组名
- `data-caption` 可选；不写时会自动取触发元素内的 `<img alt>` 或周围的 `.xx_label` / `.xx_caption` 文本
- 弹窗关闭时会自动暂停视频播放
- HTML5 视频默认带控制条 + 自动播放（由 Fancybox v5 内置实现）

---

## 三、引入顺序（已在 3 个详情页中接好）

```html
<!-- CSS：fancybox.css 必须在前 -->
<link rel="stylesheet" href="css/fancybox.css" />
<link rel="stylesheet" href="css/image-popup.css" />
<link rel="stylesheet" href="css/video-popup.css" />

<!-- JS：fancybox.umd.js 必须在前 -->
<script src="js/fancybox.umd.js"></script>
<script src="js/image-popup.js"></script>
<script src="js/video-popup.js"></script>
```

两个弹窗作用域已通过 Fancybox `mainClass` 隔离（视频弹窗作用在 `.fancybox__container.is-video-popup` 下），互不影响。
